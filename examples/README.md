# Reference implementations

Real projects built with this template. Look at these to see how the placeholders get filled in for an actual book.

- **[grandpas-book](https://github.com/jezhou/grandpas-book)** — Chinese-to-English translation of *From a Herding Boy to a Chinese Academy of Sciences Fellow*, the biography of CAS fellow Chunsheng Zhao. Demonstrates:
  - Per-page OCR'd source files with non-contiguous numbering
  - Hunanese dialect-aware translator persona
  - Photo plates with figure/figcaption HTML transformed by `split_book.py`
  - Right-aligned signature blocks (`<p align="right">`) for forewords
  - Password-protected family-only deployment

If you've used this template, open a PR adding your project here.
