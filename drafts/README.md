# Drafts

Working drafts produced during the agent loop. Use these for batches that span multiple agent passes before being merged into `OUTPUT.md`.

Naming convention: `part{N}-batch{M}-draft.md` or `part{N}-ch{A}-{B}-draft.md`.

These files are intermediate artifacts — once a batch is approved, the Senior Editor merges it into `OUTPUT.md` and the draft can be left here as history or deleted.
